# fakedata

Generate fake data for projects.

### Installation

- Install through `npm install fakedata`
- Import as `import fakedata as fake from "fakedata"`
- Use as simple as `const email = fake.email();`

### Functions

- fake.email()
- fake.username()
- fake.password()
